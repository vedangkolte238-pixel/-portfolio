🌐 Vedang Portfolio Website

📖 About

This is my personal portfolio website developed using HTML and CSS. It showcases my profile, skills, projects, education, and contact information in a clean, modern, and responsive layout.

The purpose of this website is to present my work, highlight my technical skills, and provide an easy way for recruiters and clients to connect with me.

✨ Features

* Modern and responsive design
* Clean user interface
* Home section with introduction
* About Me section
* Skills showcase
* Projects section
* Contact form
* Responsive layout for different screen sizes
* Smooth navigation

🛠️ Technologies Used

* HTML5
* CSS3
